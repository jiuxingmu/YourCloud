# 用例-组合场景模块

范围：文件夹嵌套、文件移动、文件夹移动、分享与移动联动

- `TC-049`~`TC-058`

重点边界：
- 移动弹窗可选目录与确认提交一致性
- 文件夹移动到自身/子孙目录拦截
- 分享创建后源文件移动、重命名、删除后的访问一致性
- 多级目录下文件定位与路径回显正确性

## 用例清单（组合链路）

**TC-049（P0）文件移动成功-根目录到一级目录**
- 前置条件：根目录有 `qa_move.txt`，存在目标目录 `A`
- 步骤：对 `qa_move.txt` 执行移动到 `A`
- 预期：根目录消失、`A` 内出现；接口返回成功；无重复记录

**TC-050（P0）文件移动成功-一级目录到二级目录**
- 前置条件：`A/qa_move.txt` 存在，`A/B` 已存在
- 步骤：将 `A/qa_move.txt` 移动到 `A/B`
- 预期：`A` 内文件消失，`A/B` 内可见；面包屑路径正确

**TC-051（P0）文件夹移动成功-同级目录重组**
- 前置条件：根目录存在 `A`、`B`
- 步骤：将 `B` 移动到 `A` 下
- 预期：根目录不再出现 `B`，出现 `A/B`；`B` 的子项路径级联更新

**TC-052（P0）文件夹嵌套创建成功-多级路径**
- 前置条件：用户已登录
- 步骤：依次创建 `A`、`A/B`、`A/B/C`（或通过目录内创建完成）
- 预期：三层目录均可进入；每层列表与面包屑一致

**TC-053（P0）文件夹移动失败-移动到自身**
- 前置条件：存在目录 `A`
- 步骤：对 `A` 执行移动，目标仍为 `A`
- 预期：阻断并提示非法操作；目录结构不变

**TC-054（P0）文件夹移动失败-移动到自身子目录**
- 前置条件：存在 `A/B`
- 步骤：将 `A` 移动到 `A/B`
- 预期：返回失败并提示；防止循环路径；数据不被破坏

**TC-055（P0）分享后移动-分享链接仍可访问**
- 前置条件：文件 `share.txt` 已创建分享链接（含 token）
- 步骤：将 `share.txt` 移动到其他目录后访问旧分享链接
- 预期：分享链接仍可打开并下载（按 fileId 语义）；状态为生效中

**TC-056（P0）分享后重命名/移动-分享详情信息一致**
- 前置条件：`share.txt` 已分享
- 步骤：重命名或移动该文件后进入分享页
- 预期：分享管理页与分享访问页显示当前可读信息，不出现无效引用

**TC-057（P1）分享后删除-分享链接失效语义正确**
- 前置条件：文件已分享
- 步骤：删除源文件后访问分享链接
- 预期：返回明确失效状态（如 REVOKED/NOT_FOUND 之一，需产品定义）；前端提示清晰

**TC-058（P1）组合回归-移动+嵌套+分享+回收站闭环**
- 前置条件：准备目录 `A/B/C` 与文件 `combo.txt`
- 步骤：
  1. 上传 `combo.txt`
  2. 移动到 `A/B/C`
  3. 创建分享并验证可访问
  4. 再移动到 `A`
  5. 删除并检查回收站记录
- 预期：每一步状态可追踪，列表与分享状态一致，无幽灵数据
