# YourCloud 系统性测试用例

更新时间：2026-04-29  
测试负责人：A

## 1. 测试范围与环境前提

### 1.1 测试范围

覆盖以下核心链路与风险场景：

- 账号：注册、登录、登出、会话校验（`/api/v1/auth/*`）
- 文件：上传、下载、预览缩略图、删除、移动、文件夹创建/移动
- 分享：创建分享、提取码访问、过期/失效、分享管理列表、取消分享
- 视图与状态：我的云盘、最近、星标、回收站、设置（清空搜索历史/星标/回收站）
- 权限与异常：无 token、非 owner 访问、无效 ID、无效路径、文件缺失、过期分享、错误提取码

### 1.2 不在本轮范围

- 压测、并发性能、超大文件稳定性（仅覆盖功能边界）
- 多端同步冲突、弱网断点续传

### 1.3 环境前提

- 本地环境可启动：`./scripts/dev.sh`
- 服务可用：
  - Web: `http://localhost:8082`
  - API: `http://localhost:8080`
  - 健康检查：`GET /health = 200`
- 准备账号：
  - 用户A：`qa_a@yourcloud.app`
  - 用户B：`qa_b@yourcloud.app`
  - 默认密码：`Passw0rd!`
- 准备测试文件：
  - `small.txt`（1KB）
  - `image.jpg`（100KB）
  - `video.mp4`（5MB）
  - `empty.txt`（0B）
  - `large.bin`（建议 100MB，验证大文件边界）
  - `中文 名称.txt`（中文+空格）

### 1.4 执行说明

- 每条用例均可由手工执行；涉及接口状态码时通过浏览器 Network 或 API 工具校验。
- 优先级定义：
  - **P0**：发布阻断，核心可用性/安全性
  - **P1**：主流程完整性、重要体验
  - **P2**：增强与次要边界

## 2. 系统测试用例（TC-xxx）

### 2.1 注册 / 登录 / 登出

**TC-001（P0）注册成功**
- 模块：注册
- 前置条件：未注册邮箱 `qa_new@yourcloud.app`
- 步骤：
  1. 打开登录页，切换到“注册”
  2. 输入合法邮箱与 6 位以上密码，点击“注册”
- 预期结果：
  - 前端提示注册成功并自动登录（或成功登录后进入主界面）
  - 本地存在 token

**TC-002（P0）注册失败-邮箱格式非法**
- 模块：注册
- 前置条件：无
- 步骤：输入 `abc` 与合法密码点击“注册”
- 预期结果：提示“请输入有效的邮箱地址”，不发起成功注册

**TC-003（P0）注册失败-密码长度不足**
- 模块：注册
- 前置条件：无
- 步骤：输入合法邮箱与 5 位密码点击“注册”
- 预期结果：提示“注册密码至少需要 6 位”

**TC-004（P1）注册失败-邮箱已存在**
- 模块：注册
- 前置条件：邮箱 `qa_a@yourcloud.app` 已存在
- 步骤：使用已存在邮箱再次注册
- 预期结果：返回冲突错误（`EMAIL_EXISTS`），前端展示可读错误

**TC-005（P0）登录成功**
- 模块：登录
- 前置条件：用户A已注册
- 步骤：输入正确邮箱/密码登录
- 预期结果：进入云盘主页，`/api/v1/auth/me` 返回当前用户信息

**TC-006（P0）登录失败-密码错误**
- 模块：登录
- 前置条件：用户A已注册
- 步骤：输入正确邮箱+错误密码登录
- 预期结果：返回 401（`INVALID_CREDENTIALS`），前端提示账号或密码错误

**TC-007（P0）登录失败-缺少字段**
- 模块：登录
- 前置条件：无
- 步骤：邮箱或密码留空提交
- 预期结果：前端阻断并提示“请输入邮箱和密码”

**TC-008（P0）登出成功**
- 模块：登出
- 前置条件：用户A已登录
- 步骤：点击顶部“退出”或头像菜单“退出登录”
- 预期结果：token 被清除，页面回到登录态，受保护接口不再可访问

**TC-009（P0）鉴权失败-无 token 访问文件接口**
- 模块：权限/鉴权
- 前置条件：清空 token
- 步骤：调用 `GET /api/v1/files`
- 预期结果：返回 401（未授权）

- 预期结果：分享创建成功；状态显示“生效中”；过期时间显示“永久有效”

**TC-034（P0）创建分享失败-越权文件**
- 模块：分享权限
- 前置条件：用户A知道B文件ID

**TC-035（P0）分享链接访问失败-无提取码**
- 模块：分享访问
- 前置条件：存在设置提取码的分享
- 步骤：访问 `/share/{token}` 不带 `extractCode`
- 预期结果：返回 403（`EXTRACT_CODE_INVALID`），页面提示输入提取码

**TC-036（P0）分享链接访问成功-提取码正确**

- 前置条件：分享提取码为 `9988`

**TC-037（P0）分享下载失败-提取码错误**
- 模块：分享下载
- 前置条件：存在带提取码分享
- 步骤：访问 `/api/v1/shares/{token}/download?extractCode=wrong`
- 预期结果：返回 403（`EXTRACT_CODE_INVALID`）

**TC-038（P0）分享失效-主动取消后访问**
- 模块：分享列表/取消分享
- 前置条件：用户A已创建分享
- 步骤：
  1. 在“分享管理”点击“取消分享”
  2. 再访问分享链接
- 预期结果：取消接口返回 200；访问返回 410（`REVOKED`）

**TC-039（P0）分享过期-过期后访问**
- 模块：分享访问
- 前置条件：已创建短时效分享并等待过期（或人工改过期时间）
- 步骤：访问分享详情与下载
- 预期结果：返回 410（`EXPIRED`），前端提示“分享链接已过期”

**TC-040（P1）分享列表展示与排序**
- 模块：分享管理
- 前置条件：用户A有多个分享记录
- 步骤：进入“分享管理”
- 预期结果：按创建时间倒序；显示文件名、提取码、过期时间、状态、操作按钮

**TC-041（P1）取消分享幂等**
- 模块：分享管理
- 前置条件：某分享已被取消
- 步骤：再次点击“取消分享”
- 预期结果：返回 400（`ALREADY_REVOKED`）并提示“该分享已失效”

**TC-042（P1）分享URL安全-忽略不可信Origin**
